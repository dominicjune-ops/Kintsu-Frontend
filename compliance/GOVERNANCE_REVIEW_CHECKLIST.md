# Governance Review Checklist

## Quarterly Governance Review
- [ ] Review all compliance and security policies
- [ ] Validate audit log integrity and retention
- [ ] Review RBAC and credential usage
- [ ] Check for documentation drift
- [ ] Review incident response readiness
- [ ] Update training and onboarding materials
- [ ] Review stakeholder reporting and dashboard

## Training Schedule Template
- [ ] Annual compliance and security training for all staff
- [ ] Quarterly update for privileged users
- [ ] Onboarding training for all new team members

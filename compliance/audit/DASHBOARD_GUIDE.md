# Compliance Dashboard

This script (`scripts/compliance_dashboard.py`) generates a summary dashboard of audit log activity and documentation status for stakeholders.

## Usage

```
python scripts/compliance_dashboard.py
```

- Outputs a markdown report to `/compliance/audit/compliance-dashboard.md`.
- Run this as a scheduled job (e.g., weekly or monthly) to keep stakeholders informed.
